# DentVantage Agent Hub

**AI Empowering Dentists** — živý Agent Hub pro DentVantage platformu.

## Soubory

- `index.html` — Agent Hub (hlavní stránka)
- `DentVantage_FinancniAgent_v3_HYBRID.html` — Finanční Agent dashboard

## Deploy na GitHub Pages

1. Nahraj tento repozitář na GitHub
2. Settings → Pages → Source: `main` branch, root `/`
3. Za pár minut bude live na `https://[tvuj-username].github.io/dentvantage-hub/`

## Deploy na Netlify Drop

1. Jdi na netlify.com/drop
2. Přetáhni složku nebo `index.html`
3. Okamžitě live URL

## Tech

- Single-file HTML, žádné závislosti
- Logo + Avatar embedded jako base64
- YouTube video embed (Shorts jFRtZuoxMSk)
- FA Dashboard: 16 procedur, reálná data
- Buddy Link Generator s JWT tokenem
